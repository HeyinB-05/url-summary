库

这里收集一些使用过的库。

前端常用

  • decimal.js 一个用于任意精度十进制算术的 JavaScript 库。**big.js、bignumber.js 和 decimal.js 的区别 **

工具库

  • 正则大全
  • Splitpanes 拆分窗格插件些使用过的库。
  • webcam 访问摄像头 上传到服务器。
  • nodemailer 发送邮件。
  • vxetable 基于 vue 的表格库。
  • turnjs/ 使用 HTML5 制作翻书
  • Lena.js一款轻量级的 JavaScript 图像处理库 学习链接
  • driver.js 官网 一个用于引导用户浏览网站的库。
  • wangEditor 5 开源 Web 富文本编辑器，开箱即用，配置简单
  • NPlayer NPlayer 是由 Typescript 加 Sass 编写，无任何第三方运行时依赖，兼容 IE11，支持移动端、支持 SSR、支持直播的播放器。
  • compressorjs 纯前端的图片压缩工具效果演示
  • scrollrevealjs 瀑布流页面滚动动画插件
  • Magic Grid 瀑布流布局插件，自适应九宫格
  • fullcalendar 日历插件
  • html2canvas 将 html 转为图片 相关解决方案
  • printjs 打印插件
  • Fabric.js 文档 Canvs的js库：Fabric.js简单强大，用于绘制各种图形
  • chinese-days 中国节假日
  • tiffjs tiffjs 是一个用于解析和解码 tiff 图像的 JavaScript 库。

Vue 基础

  • VueUse 基础 Vue 组合式工具的集合，功能强大的 vue 函数库，提供了多种 hooks。
  • Vue.js 挑战 Vuejs 挑战平台

Excel

  • SheetJS 基功能强大的 excel 库，支持导入导出。
  • exceljs 基于 SheetJS 的 excel 库，支持导入导出，功能更强大。
  • xlsxjs 轻量级 excel 处理文件。
  • X-Spreadsheet 基于 canvas 的 excel 库，可以在网页显示 excel 文件。

可视化

  • ECharts 百度开源的图表库，功能强大，支持多种图表类型。
  • Three.js 3D 图形库，功能强大，支持多种 3D 图形类型。
  • DataV 大屏数据可视化库，支持多种图表类型。

Node

  • getlocalip 这是一个获取本地 ip 的节点线工具。
  • mcprotocolmcprotocol 是一个库，允许使用三菱记录的 MC（MELSEC 通信）以太网协议与三菱 PLC（当前仅测试 FX3U）进行通信。警告 - 这是测试版代码

  • tsParticles 一个用于创建粒子的轻量级 TypeScript 库。

  • Waves.jsWaves.js 入门指南：让文字动画效果更加出色
  • Animate.css 动画库，可以用 js 控制动画。

  • vant vant4
