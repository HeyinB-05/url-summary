库

这里收集一些使用过的库。

前端常用

  • 正则大全
  • Splitpanes 拆分窗格插件些使用过的库。

工具库

  • webcam 访问摄像头 上传到服务器。

Vue 基础

  • VueUse 基础 Vue 组合式工具的集合，功能强大的 vue 函数库，提供了多种 hooks。
  • Vue.js 挑战 Vuejs 挑战平台

Excel

  • SheetJS 基功能强大的 excel 库，支持导入导出。
  • exceljs 基于 SheetJS 的 excel 库，支持导入导出，功能更强大。
  • xlsxjs 轻量级 excel 处理文件。
  • X-Spreadsheet 基于 canvas 的 excel 库，可以在网页显示 excel 文件。

可视化

  • ECharts 百度开源的图表库，功能强大，支持多种图表类型。
  • Three.js 3D 图形库，功能强大，支持多种 3D 图形类型。
  • DataV 大屏数据可视化库，支持多种图表类型。

Node

  • getlocalip 这是一个获取本地ip的节点线工具。

动画

  • animejs 动画库，可以用js控制动画。
  • tsParticles 一个用于创建粒子的轻量级TypeScript库。

CSS

  • tailwindcss 原子化CSS，只需写class，无需写样式。

一些大佬

  • 网站汇总 网址合集
  • panjiachen 一个字节跳动的大佬
  • yunyoujun 云游君，elememt-plus开发者之一
  • 神奇海螺试验场
  • wangyasai 一个3D网站
